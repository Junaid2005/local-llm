# Local LLM Project

Starter Python project layout.

## Quick start

1) Create a virtual environment

```bash
python -m venv .venv
```

2) Run via the JupyterHub helper script (required for now)

```bash
./scripts/run_model.sh
```

JupyterHub URL (QMUL):

```
https://hub.comp-teach.qmul.ac.uk/user/{username}/proxy/8501/
```
